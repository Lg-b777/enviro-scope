== Description ==
Enviroscope is a lightweight WordPress plugin designed to "sniff" your environment and display key information like PHP version, MySQL version, server details, active theme, and installed plugins.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate it via the Plugins menu in WordPress.
3. Navigate to the admin dashboard to view system information.
